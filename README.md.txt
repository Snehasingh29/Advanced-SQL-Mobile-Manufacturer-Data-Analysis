# 📱 Advanced SQL - Mobile Manufacturer Data Analysis

## 📌 Project Overview

This project demonstrates advanced SQL data analysis on a Mobile Manufacturer dataset using SQL Server Management Studio (SSMS). The objective is to solve real-world business problems by writing optimized SQL queries and extracting meaningful business insights.

## 🎯 Objectives

- Analyze mobile manufacturer data.
- Solve business case study questions using SQL.
- Generate meaningful business insights.
- Improve data analysis and query writing skills.

## 🛠️ Tools Used

- SQL Server Management Studio (SSMS)
- Microsoft SQL Server

## 📚 SQL Concepts Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- JOINS (INNER, LEFT, RIGHT)
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- CASE WHEN
- String Functions
- Date Functions
